# Retrieval Protected Phrases Review

This file is for manual review of the runtime `protected_phrases` allowlist.

Principles:
- Runtime `protected_phrases` are allowlist-only.
- Single-word terms should not be protected here.
- Generic headings and structural labels stay in `extraction_candidates.json`, not in runtime protection.
- Automated extraction is a discovery tool, not a runtime authority.
